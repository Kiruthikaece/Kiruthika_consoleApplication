## Library Management System

A library management system is software that is designed to manage all the functions of a library. It helps librarian to maintain the database of new books and the books that are borrowed by members along with their due dates. This system completely automates all your libraryâ€™s activities. A library management system is used to maintain library records.

## Tech Stack

Java

## Type of application

Business to customer

## Features
- Admin user Login / Logout 
- Libray setup with the basic details.
- Add new books to the library
- New account creation for customer
- Search and showing a books list to the user.
- Assign books to the user with the return date.
- Return process and overdue handling.
- Managing the books.
- Managing the user account
